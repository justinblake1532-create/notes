
# Types of Network Attacks


## Credential Harvesting
 : A type of reconaissance to learn passwords or cyrptographic secrets to obtain authenticated access to network systems


## Denial of Service(DoS)
 : An attack that causes a host system or service to become unavailable


## Weaponization, Delivery, and Breach
 : Attacks that allow threat actors to gain unauthorized access without having to authenticate; using malicious code to exploid vulnerabilites in services


## Command and Control(C&C)
 : Using malicious code to compromise a remote host and maintain access to it for a long period of time


## Reflected Attack
 : Spoofing the targets IP addresses and attempts to make network connections with third-party devices


## Amplified Attack
 : An attack that targets weaknesses in applications protocols to consume more of the target's bandwidth


## On-Path Attack
 : An attack where a threat actor gains access to a network connection and discretely monitors the connection


## Address Resolution Protocol(ARP) Poisioning
 : An attack that sends false ARP packets over a network to link the threat actors MAC address to a legitimate network's IP address
