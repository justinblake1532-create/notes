
# Coding Errors


## Compile
 : An error that occurs during the building or compilation stage; prevents applications from running


## Runtime
 : An error that occurs when the application is running


## Exceptions
 : How an application responds to unexpected errors that can lead to holes in teh security of the application
 * Exceptions must be controlled <ins>well</ins> in the application
