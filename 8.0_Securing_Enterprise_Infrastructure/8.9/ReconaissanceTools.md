
# Reconaissance Tools


## * OSINT
## * theHarvester
 : Passive recon tool to gather information on a target from public resources(emails, names, subdomains, IP addresses, etc.)
## * Shodan
 : A search engine that can query information on targets to see if there are any known vulnerabilities with that target
## * Dnsenum
 : A program that can search for DNS related information(usernames, system names, IP addresses, etc.)
## curl and wget
 : Command-Line tools to download and upload files
 - Download and entire website
## Scanless
 : Users exploitation websites to perform port scans rather than a traditional port scanner
## Sn1per 
 : An automated scanner to enumerate and scan for vulnerablilites; can find DNS information, open ports, running services, etc.
## Nessus
 : A proprietary vulerability scanner to scan a network for vulnerabilities
