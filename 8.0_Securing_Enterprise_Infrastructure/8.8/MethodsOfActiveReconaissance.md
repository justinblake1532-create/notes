
# Methods of Active Reconaissance


## Port Scanning
 : Scanning a target network to find vulnerable netowrk ports to exploit


## Service Enumeration
 : Interacting with identified services to find more detailed information


## OS Fingerprinting
 : Identifying the OS running on the target system(s)


## DNS Enumeration
: Gathering detailed information about the target's DNS infrastructure(domain names, subdomains, IP addresses)


## Web Application Crawling
 : Browsing public sites from the target to find any vulnerabilities
