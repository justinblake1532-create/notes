
# Password Attacks


## Online Attack
 : An attack where the threat actor engages with the authentication mechanisms to exploit them


## Offline Attack
 : The threat actor has obtained a database of password hashes from a targets system and attempts to crack the database


## Password Spraying
 : A threat actor uses the same password across multiple user accounts


## Dictionary Attack
 : A threat actor uses a common list of passwords to guess the password
