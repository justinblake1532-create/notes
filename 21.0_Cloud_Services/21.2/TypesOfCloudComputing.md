
# Types of Cloud Computing


## Public Cloud
 : Resources can be accessed by anyone; resources are generally free to use

## Private Cloud
 : Resources can only be accessed by a single organization/entity; data is kept secret

## Hybrid Cloud
 : Combination of different cloud models into one model; helps expand the functionality of the services offered in the cloud

## Community Cloud 
 : Resources are shared by multiple organizations; generally hosted by a third-party organization
